# 三格式差异化深度指南

## 核心认知：三种不同的"观众契约"

每种格式在开播前就与观众签订了一份隐性契约：

- **竖屏短剧**："我保证每2分钟给你一次爽"
- **20分钟剧集**："我保证每集给你一个完整的情绪旅程"
- **院线电影**："我保证这90分钟会改变你看某件事的方式"

**改编的本质，就是重新签订契约。**

---

## 一、叙事时间差异

### 短剧的时间逻辑
- 现实感极弱，一天内可以发生10件大事
- 时间是情节的容器，不是叙事工具
- 观众不在乎"这怎么可能发生"，只在乎"爽不爽"

### 剧集的时间逻辑
- 需要时间的"可信度"，但仍可压缩
- 时间流逝是人物成长的外在标记
- 季与季之间可以有时间跳跃

### 电影的时间逻辑
- 时间是主题的一部分（《请回答1988》《爱乐之城》）
- 非线性叙事、倒叙都有意义
- 每一个时间节点的选择都是导演的表达

---

## 二、人物设计差异

### 短剧人物公式
```
主角 = 标签 + 金手指
反派 = 阻碍工具 + 嚣张系数
配角 = 助攻 or 炮灰
```

### 剧集人物公式
```
主角 = 欲望 + 障碍 + 弱点（成长空间）
反派 = 自己的动机 + 扭曲的逻辑
配角 = 独立的小故事线 + 与主角主题呼应
```

### 电影人物公式
```
主角 = 外部目标 + 内在需求（两者冲突）+ 主题缺陷
反派 = 主角内心阴暗面的外化
配角 = 主题的不同侧面
```

**关键问题：**
- 短剧问："他们能赢吗？"
- 剧集问："他们会成长吗？"
- 电影问："他们会改变吗？"

---

## 三、对白差异

### 短剧对白特征
```
反派：你算什么东西！
主角：（金手指激活）让我来告诉你我是什么东西。
[身份揭露，打脸完成]
```
特点：功能性，直白，即时满足，信息量密集

### 剧集对白特征
```
A：你真的要走？
B：（沉默片刻）你希望我留下吗？
A：（转身，不看他）我希望你自己想清楚。
```
特点：有潜台词，一句话有两个意思，为下集埋线

### 电影对白特征
```
A：我找到他了。
B：（看着A，慢慢说）然后呢？
A：（停顿，这是他第一次真正想清楚）
   然后我要告诉他，我原谅他了。
   （更小声）不是为了他。
```
特点：极度精炼，沉默是对白，台词改变人物

## 四、冲突层次差异

| 层次 | 短剧 | 剧集 | 电影 |
|------|------|------|------|
| 外部冲突 | ★★★★★ | ★★★★ | ★★★ |
| 人际冲突 | ★★ | ★★★★ | ★★★★ |
| 内在冲突 | ★ | ★★★ | ★★★★★ |
| 社会冲突 | ★ | ★★★ | ★★★★ |

---

## 五、结局要求差异

### 短剧结局
- 爽，越爽越好
- 开放结局为续集服务
- 不需要主角"改变"，需要主角"赢"

### 剧集结局（季末）
- 主线有阶段性收束
- 人物关系有重大变化
- 留下足够的续集钩子
- 主角有可见的成长

### 电影结局
- 主题必须得到清晰表达（正面、负面、开放均可）
- 主角的内在改变必须可见
- 最后一个画面是主题的视觉化
- 观众走出影院时应该"有什么不同了"

---

## 六、差异化的实际操作检验

改编完成后，用以下问题检验格式适配性：

**检验剧集改编**
- [ ] 去掉所有打脸场景，人物关系还成立吗？
- [ ] 副线和主线主题是否呼应，还是只是填充？
- [ ] 每集末尾的钩子类型是否多样，不只是悬念？
- [ ] 主角在第3集和第8集有什么不同？

**检验电影改编**
- [ ] 这个故事的"主题一句话"是什么？
- [ ] 主角在第一幕末和第三幕末的内心有何不同？
- [ ] 最后一场戏的最后一个画面，是否呼应了第一场戏？
- [ ] 如果把所有对白删掉，视觉叙事还能讲清楚主线吗？
