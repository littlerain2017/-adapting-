# 院线电影剧本格式规范

## 一、页面设置

- 字体：Courier New 12pt（英文标准）；中文用等宽字体
- 页边距：上下各1英寸，左1.5英寸，右1英寸
- 每页约1分钟成片时长（经验值）
- 标准院线剧本：90-120页

---

## 二、格式元素

### 场景行（Scene Heading / Slug Line）

**格式**：`内/外景. 地点 - 时间`

```
INT. 废弃工厂 - 夜
EXT. 北京长安街 - 清晨
INT./EXT. 行驶中的出租车 - 日（内外兼有）
```

规则：
- 全大写
- 每个新场景必须有场景行
- 地点要具体，但不要过度描述
- 时间只写：日、夜、稍后、连续

---

### 动作行（Action / Description）

**格式**：左对齐，段落之间空行

```
陈默站在窗边，背对着房间。窗外，北京的灯火密集却遥远。

她的手机震动。她没有看。
```

规则：
- 现在时，主动语态
- 只写"镜头能看到的"，不写内心活动
- 每段不超过4行，超过就分段
- 动词优先，避免形容词堆砌
- 禁止摄影机指令（"推镜头"、"特写"等）——除非你是导演兼编剧

**强场景描述示例**：
```
差：林晓很伤心地站在雨中，感到十分孤独。
好：林晓站在雨中。她没有撑伞。
```

---

### 对白（Dialogue）

**格式**：
```
                    角色名
          台词内容（左缩进约3.5英寸）
```

带括注时：
```
                    林晓
          （转身，看着他）
          你来晚了。
```

规则：
- 角色名全大写，居中偏右
- 括注（parenthetical）只在动作影响台词含义时使用，不要滥用
- 台词不要超过3-4行；超过说明信息太密，需要拆分或用动作替代
- 禁止"As you know, Bob"式台词（角色向对方解释对方已知的信息）

---

### 过渡（Transition）

只在功能性过渡时使用，不要装饰性使用：

```
CUT TO:（硬切，制造冲击）
DISSOLVE TO:（溶入，表示时间流逝）
SMASH CUT TO:（强硬切，制造反差）
MATCH CUT TO:（匹配剪辑，视觉或主题呼应）
```

现代剧本中，大多数过渡可以省略。

---

### 插入/字幕

```
SUPER: "2019年，北京"
INSERT: 林晓手机屏幕上的短信
BACK TO SCENE
```

## 三、结构节拍表（Blake Snyder Beat Sheet）

| 节拍 | 页码 | 说明 |
|------|------|------|
| 开场画面 | 1 | 建立世界基调的视觉 |
| 主题陈述 | 5 | 有人说出主题（通常是配角对主角说） |
| 铺垫 | 1-10 | 主角的日常世界和内心缺陷 |
| 触发事件 | 12 | 打破主角日常的事件 |
| 讨论/拒绝 | 12-25 | 主角犹豫，被推向行动 |
| 进入第二幕 | 25 | 主角做出选择，踏上旅程 |
| B故事 | 30 | 副线开始（通常是爱情线，承载主题） |
| 玩耍时间 | 30-55 | 主角在新世界中探索，展示电影的核心概念 |
| 中点 | 55 | 假胜利或假失败，提升赌注 |
| 坏人逼近 | 55-75 | 外部压力加剧，主角的方法开始失效 |
| 一切崩溃 | 75 | 最大的失败，副线也出问题 |
| 灵魂黑夜 | 75-85 | 主角最低谷，内心缺陷彻底暴露 |
| 进入第三幕 | 85 | 主角获得顿悟，凭借内在改变 |
| 高潮 | 85-110 | 最终决战，主题得到验证 |
| 结局画面 | 110 | 与开场画面形成对比，显示世界改变了 |

## 四、院线剧本与短剧/剧集的格式对比

| 元素 | 院线剧本 | 剧集剧本 | 短剧 |
|------|----------|----------|------|
| 场景行 | 必须，规范 | 必须，规范 | 简化，常省略时间 |
| 动作行 | 极简，视觉化 | 简洁 | 可以更口语 |
| 对白格式 | 严格居中缩进 | 严格 | 可灵活 |
| 摄影机指令 | 禁止 | 禁止（除pilot） | 常见 |
| 页数/时长 | 1页≈1分钟 | 1页≈1分钟 | 无严格要求 |
| 幕标记 | ACT ONE/TWO/THREE（可选）| ACT ONE/TWO/TAG（必须）| 无 |
| 章节/集标记 | 无 | END OF ACT ONE | 集数标记 |

---

## 五、中文院线剧本特殊说明

中文剧本无统一行业标准，但专业制作公司普遍遵循：
- 场景行：`内景/外景. 地点 时间`（或直接写"室内·地点·时间"）
- 动作行：第三人称，现在时，简洁
- 对白：直接写角色名加冒号，或仿好莱坞格式
- 大多数院线级项目会要求按好莱坞格式排版（方便国际合拍）
