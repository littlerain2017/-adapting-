# 20分钟剧集格式规范与分集结构模板

## 一、20分钟剧集的结构特点

20分钟剧集（常见于流媒体、网剧、情景喜剧）通常分为两幕或三幕，每幕有明确的情绪转折点。

### 标准两幕结构（含TAG）

```
片头片尾字幕前：冷开场（Cold Open）1-2分钟
  └ 直接切入事件，不解释背景，制造即时钩子

ACT ONE（第一幕）7-8分钟
  └ 建立本集核心冲突
  └ 主线事件触发
  └ 幕末：转折点，逼出"不可逆选择"

ACT TWO（第二幕）8-9分钟
  └ 冲突激化，主副线交织
  └ 中点：假解决或更大危机
  └ 幕末：高潮 + 本集情绪释放

TAG（尾声）1-2分钟
  └ 情绪沉淀，为下集埋钩子
  └ 可以是笑点、温情点、或悬念点
```

## 二、分集结构模板

### 单集内部节拍（20分钟版）

| 时间点 | 节拍名称 | 功能 |
|--------|----------|------|
| 0:00-1:30 | 冷开场 | 钩子，直接抛出问题或冲突 |
| 1:30-5:00 | 建立本集状态 | 人物当前状态，本集矛盾种子 |
| 5:00-8:00 | 触发事件 | 打破本集平衡的具体事件 |
| 8:00-10:00 | 第一幕末 | 主角被迫做选择，进入困境 |
| 10:00-14:00 | 冲突激化 | 主线深化，副线插入 |
| 14:00-17:00 | 最黑暗时刻 | 外部压力最大，主角最被动 |
| 17:00-19:00 | 转折/高潮 | 主角主动出击，情绪释放 |
| 19:00-20:00 | TAG/钩子 | 本集结，下集预埋 |

## 四、主副线设计原则

### 主线
- 每集推进一个具体的外部目标
- 有明确的"本集成败"，不要每集都悬而未决

### 副线
- 副线的主题要与主线呼应（不是装饰，是镜像）
- 副线可以在主线最黑暗时提供情绪喘息
- 副线每集出现1-2次，不抢主线节奏

### 主副线交织节奏示例

```
0-5分  主线：触发事件
5-7分  副线：建立副线状态
7-10分 主线：深化冲突
10-12分 副线：副线小转折
12-16分 主线：高潮前积累
16-18分 主线：高潮
18-20分 副线+主线：双线收束，TAG
```

## 五、集间钩子类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 悬念钩子 | 抛出未解决的问题 | 悬疑、职场剧 |
| 关系钩子 | 人物关系发生微妙变化 | 爱情、家庭剧 |
| 反转钩子 | 揭示令人意外的信息 | 剧情、悬疑剧 |
| 情绪钩子 | 留下强烈情绪余韵 | 情感剧 |
| 动作钩子 | 主角做出下一步行动 | 动作、冒险剧 |

## 六、从短剧集数到20分钟剧集的转换比例

| 短剧集数 | 对应20分钟剧集集数 | 说明 |
|----------|-------------------|------|
| 10集短剧 | 约3集 | 每3集短剧≈1集20分钟 |
| 30集短剧 | 约10集 | 适合1季网剧 |
| 80集短剧 | 约24-30集 | 可分2季 |

注意：这是情节密度的粗略对应，实际改编需要大量扩充人物深度和副线。
